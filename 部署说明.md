## 🎯 Vercel部署步骤
1. 访问 https://vercel.com
2. GitHub登录
3. 导入 `jetzhao-portfolio` 仓库
4. 确认配置：
   - Framework: Vue.js
   - Build Command: `npm run build`
   - Output Directory: `dist`
5. 点击Deploy
